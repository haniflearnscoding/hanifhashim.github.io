3D Tree

Procedural 3D tree modeling and animation in Blender

3D Blender Animation

About This Project

This project explores procedural modeling techniques in Blender to create a realistic 3D tree. The tree was modeled using Blender's geometry nodes and shader systems to achieve natural-looking foliage and organic branch structures.

Technical Details

  • Software: Blender 3D
  • Techniques: Procedural modeling, geometry nodes
  • Rendering: Cycles/Eevee
