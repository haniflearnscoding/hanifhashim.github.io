Venice Apartment

PS2 Asset Extraction and Unreal Engine Recreation

Role 3D Artist & Technical Artist
Year 2025
Technologies Unreal Engine, PS2 Emulation, Asset Extraction

Concept

A technical art project recreating the iconic Venice apartment from Sly Cooper 3: Honor Among Thieves by extracting original textures and models from PS2 emulation and rebuilding the environment in Unreal Engine with modern rendering techniques.

Challenge

How do you preserve and reimagine a beloved video game environment from 2005? How can you extract assets from a PS2 game's proprietary format and translate them into a modern game engine while honoring the original art direction?

Solution

Using PS2 emulation tools, I extracted original textures and 3D models from Sly Cooper 3: Honor Among Thieves, then rebuilt the Venice apartment safehouse environment in Unreal Engine. The project involved reverse-engineering PS2 asset formats, cleaning and optimizing legacy geometry, and applying modern lighting, materials, and post-processing to create a faithful yet enhanced recreation of this nostalgic space.

The Venice apartment safehouse recreated with Unreal Engine's lighting and rendering

Technical Process

PS2 Emulation & Extraction

Asset Conversion

About Sly Cooper 3

Sly Cooper 3: Honor Among Thieves (2005) is a beloved PlayStation 2 stealth platformer developed by Sucker Punch Productions. The Venice safehouse serves as the crew's headquarters during the Italian heist operation—a richly detailed interior space filled with ornate furniture, artwork, and period details that establish the game's charming art-deco-meets-cartoon-noir aesthetic.

This environment represents a technical achievement for PS2 hardware: multiple light sources, reflective surfaces, detailed textures, and atmospheric depth all running on 32MB of RAM. Recreating it in Unreal Engine pays homage to this technical artistry while exploring what these spaces might look like with modern rendering capabilities.

Lighting & Atmosphere

The scene captures the Venice apartment's signature moody atmosphere: warm spotlight illuminating the golden throne-like chair, purple and burgundy upholstery catching volumetric light, rich wood paneling, and dramatic curtains framing the space. Unreal Engine's real-time lighting allows for dynamic shadows and bounce light that enhance the depth and warmth of the original design.

Careful attention was paid to preserving the stylized, slightly exaggerated proportions and color choices of the original while taking advantage of modern techniques like screen-space reflections, ambient occlusion, and volumetric fog to add richness without losing the game's distinctive visual character.

Asset Extraction Workflow

The extraction process involved running Sly Cooper 3 through a PS2 emulator with debugging tools enabled, allowing access to the game's memory and file structure. Textures were exported from the game's proprietary TIM2 format, while models required parsing custom geometry formats used by Sucker Punch's engine.

Many PS2-era models used vertex colors and simple UV mapping optimized for the console's limited texture memory. These techniques needed translation into modern PBR material workflows, creating base color, roughness, and normal maps that respect the original aesthetic while taking advantage of Unreal's material system.

Preservation & Nostalgia

This project explores game preservation through technical reconstruction. As original PS2 hardware ages and becomes less accessible, extracting and recreating these environments ensures they can be experienced by future audiences. The Venice apartment represents countless hours players spent between heists, planning missions, and enjoying character interactions—it's a space rich with nostalgia and emotional significance.

By bringing this environment into Unreal Engine, the project also opens possibilities for interactive experiences, VR exploration, or further artistic reinterpretation while maintaining a connection to the source material.

Technical Challenges

PS2 games used highly optimized, non-standard asset pipelines that don't map directly to modern engines. Texture compression, vertex color baking, and custom shader approximations all required manual interpretation and recreation. The original game ran at 512x448 resolution—translating these assets to 4K rendering revealed texture limitations that needed careful upscaling or recreation.

Lighting posed particular challenges: the PS2 version used pre-baked lightmaps and vertex lighting, while Unreal Engine uses real-time global illumination. Recreating the mood required studying the original lighting setup and translating it into dynamic light sources that capture the same atmosphere.

Reflection

This project sits at the intersection of technical art, game preservation, and nostalgia. It demonstrates skills in reverse engineering, asset pipeline development, and environment art while paying tribute to the craftsmanship of PS2-era game developers who created rich, memorable spaces within severe hardware constraints.

The Venice apartment in Sly Cooper 3 represents a specific moment in game design history—before HD graphics, before PBR materials, when every texture and polygon had to be meticulously optimized. Recreating it in Unreal Engine bridges past and present, showing how these beloved spaces can evolve while retaining their essential character.
